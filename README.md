# Vacation Planner — Bariloche 2026

Planificador interactivo de viaje para el viaje a Bariloche (14–18 agosto 2026).

## Qué hace

- 6 secciones: Resumen, Vuelos, Hospedaje, Ski & Equipo, Actividades, Traslados
- Comparar y seleccionar opciones por categoría con checkboxes
- Totales automáticos separados por persona (Santi desde MDZ / Amigo desde BUE)
- Agregar, editar y eliminar ítems via modal
- Estado persistido en `localStorage`
- Light mode + dark mode automático

## Stack

HTML + CSS + JS vanilla, un solo archivo `index.html`. Sin frameworks ni dependencias excepto [Tabler Icons](https://tabler.io/icons) vía CDN.

## Cómo usar

Abrir `index.html` directamente en el navegador. No requiere servidor ni instalación.
